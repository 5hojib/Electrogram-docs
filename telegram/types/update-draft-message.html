    class pyrogram.raw.types.UpdateDraftMessage

    Notifies a change of a message draft.

    Constructor of Update.

    Details:

    • Layer: 223

    • ID: EDFC111E

    Parameters:

    • peer (Peer) – The peer to which the draft is associated

    • draft (DraftMessage) – The draft

    • top_msg_id (int 32-bit, optional) – ID of the forum topic to which the draft is associated

    • saved_peer_id (Peer, optional) – If set, the draft is related to the specified monoforum topic ID ».
