    UpdateShortSentMessage

    class pyrogram.raw.types.UpdateShortSentMessage

    Shortened constructor containing info on one outgoing message to a contact (the destination chat has to be extracted from the method call that returned this object).

    Constructor of Updates.

    Details:

    • Layer: 222

    • ID: 9015E101

    Parameters:

    • id (int 32-bit) – ID of the sent message

    • pts (int 32-bit) – PTS

    • pts_count (int 32-bit) – PTS count

    • date (int 32-bit) – date

    • out (bool, optional) – Whether the message is outgoing

    • media (MessageMedia, optional) – Attached media

    • entities (List of MessageEntity, optional) – Entities for styled text

    • ttl_period (int 32-bit, optional) – Time To Live of the message, once message.date+message.ttl_period === time(), the message will be deleted on the server, and must be deleted locally as well.
